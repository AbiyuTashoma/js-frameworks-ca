- Dynamic search bar

- Contact form

## Built With

- React app JS framework
- Zustand state management
- React bootstrap
- Yup form validation
- SCSS

## Running

To get started, follow one the following options

1. visit https://cosmic-monstera-e885b3.netlify.app/

2. clone the repo:

```
https://github.com/AbiyuTashoma/js-frameworks-ca
```

and then

```
- npm install
- npm run start
```
