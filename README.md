## Getting Started

To get started, follow one the following options

### Visit the hosted website

https://cosmic-monstera-e885b3.netlify.app/

### Installing

1. Clone the repo:

```
https://github.com/AbiyuTashoma/js-frameworks-ca
```

2. Install the dependencies:

```
- npm install
```

3. running:

```
- npm run start
```

3. testing:

```
- npm run test
```
